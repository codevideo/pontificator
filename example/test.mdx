Things like these markdown markups should be removed:

# Heading 1

## Heading 2

### Heading 3

_italics_

*bold*

Emojis should also be removed

emojis: 😎, 🍺 - should not be able to see these anymore

custom components in MDX should also be removed in the resulting txt:

<SomeCustomComponent/>

Links should only keep their label:

[some link](https://www.google.com)

Codeblocks should also be removed:

```javascript
console.log('hello world');
```
